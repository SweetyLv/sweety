## cmd & npm & node
* cd ..  返回上一层目录
* cd \ 返回根目录
* npm -v  看npm版本
* node -v 看node版本
* npm install npm -g 通过npm命令升级为npm的新版本for Windows
* npm install <Module Name>  通过npm安装Node.js模块语法
